### 예제4.  R의 자료구조 5가지 ###

■ ** R의 자료구조 **
- vector: 같은 데이터 타입을 갖는 1차원 배열 구조
- matrix: 같은 데이터 타입을 갖는 2차원 배열 구조
- array: 같은 데이터 타입을 갖는 4차원 배열 구조
- data frame: 서로 다른 데이터 타입을 갖는 2차원 배열 구조(오라클의 테이블)
- list: 서로 다른 데이터 구조(vector, matrix, array, dataframe)인 데이터 타입이 중첩된 구조
