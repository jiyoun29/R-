■ 문제 18. 직업이 SALESMAN인 사원들의 부서번호를 출력하는데, 중복을 제거해서 출력하세요!

```r
d <- emp[ grep("SALESMAN", emp$job), c("deptno")]
unique( d ) # 틀림 ㅠㅠ

result <- emp[(emp$job == "SALESMAN"), c("deptno")] # [] 사용법 유의!
unique( result )
```
